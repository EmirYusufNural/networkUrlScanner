 Project Content
🚀 Targeted Features:
✅ Scan websites and identify entry points (forms, URL parameters)
✅ Testing SQL Injection, XSS, SSTI and IDOR vulnerabilities
✅ Get the target URL from the user and perform automatic scanning
✅ Output the screening results as a report
_______________________________________________________________________________________
Main Operating Logic
1️⃣ Targeting: Obtains the target URL from the user.
2️⃣ Finding Entry Points: Forms, GET and POST parameters are determined.
3️⃣ Open Scan Module:

SQL Injection test FIX
XSS testing
SSTI test
IDOR test
4️⃣ Reporting Results: If vulnerabilities are found, do not save the details in a file.
